# Auto Compose

This project is designed to address to [NetAI, GIST](https://netai.smartx.kr/).

## Usage

### Simple Task

```bash
./config.py -f config.yaml
```

### Multiple Tasks

```bash
./batch.py -f batch.yaml -s settings.yaml --verbose
```

### Visualize Tasks

```bash
./visualize.py rook -g
```

## Requirements

### Arch

```bash
./requirements-arch.sh
```
