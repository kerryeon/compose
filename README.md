# k8s compose

This project is designed to address to [NetAI, GIST](https://nm.gist.ac.kr/).

## Usage

### Simple Task

``` bash
./config.py config.yaml
```

### Multiple Tasks

``` bash
./batch.py batch.yaml
```

### Visualize Tasks

``` bash
./visualize.py rook
```

## Requirements

### Arch

``` bash
./requirements-arch.sh
```
